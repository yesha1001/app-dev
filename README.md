# Stranger Things

**Stranger Things** is a science fiction horror series created by the Duffer Brothers.  
It premiered on **Netflix** in 2016.

## Main Characters
- Eleven (Millie Bobby Brown)
- Mike Wheeler (Finn Wolfhard)
- Dustin Henderson (Gaten Matarazzo)
- Lucas Sinclair (Caleb McLaughlin)
- Will Byers (Noah Schnapp)

## Setting
The story is set in a small town called *Hawkins, Indiana*, during the 1980s.

> "Friends don't lie." – Eleven

## Why I Like It
1. Amazing 80s vibe
2. Strong character development
3. Mysterious and thrilling plot
